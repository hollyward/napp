Install nodeJS
nodejs.org

Installing Express
$ npm init
  creates package.json
$ npm install express --save
   installs express and adds express to package.json
$ node app
    runs app
    
$ npm install nodemon
    runs app without restarting

$ npm install ejs --save
    embedded javaScript template engine
